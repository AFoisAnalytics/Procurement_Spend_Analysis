# Procurement Spend Analysis

Database: WildWorldImportersDW2017

Measures Created in DAX (POs = Purchase Orders):

    - Total $ Purchases
	  - Total N. of POs
	  - Total Ordered Quantity
	  - Total Final POs
	  - Total Non Final POs
	  - Ordered Outers
	  - Received Outers
